# Tower of Hanoi

This repository contains an implementation of the **Tower of Hanoi** puzzle, a classic problem in computer science used to demonstrate recursion and algorithmic thinking.

The objective is to move all disks from the source peg to the destination peg using an auxiliary peg while following strict rules.

---

## Problem Statement

* Only one disk can be moved at a time
* A disk can only be placed on top of a larger disk or an empty peg
* All disks start on the source peg in decreasing size order

---

## Features

* Recursive solution (core logic)
* Iterative approach (optimized understanding)
* Step-by-step move generation
* Supports customizable number of disks
* Simple visualization support (if using HTML)

---

## Tech Stack

* **Language:** Python
* **Concepts Used:** Recursion, Stack Logic, Algorithm Design

## Getting Started

### Prerequisites

* Python 3.x installed on your system

---

### Installation

```bash
git clone https://github.com/anshu-2005/tower-of-hanoi.git
cd tower-of-hanoi
```

---

## Usage

### Run the Program

```bash
python tower_of_hanoi.py
```

Modify the number of disks directly in the script to test different scenarios.
